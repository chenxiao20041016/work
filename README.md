# RPN 计算器

一个基于逆波兰表示法（Reverse Polish Notation）的命令行计算器，使用 C++ 编写。

## 功能特点

- 支持基础四则运算（加、减、乘、除）
- 完全支持负数和浮点数运算
- 友好的错误提示信息
- 简洁的逆波兰表示法输入

## 编译指南

### 环境要求

- 支持 C++11 或更高标准的 C++ 编译器（如 g++）

### 编译步骤

1. 将源代码保存为 `rpn_calc.cpp` 文件
2. 打开终端，导航到源代码所在目录
3. 执行以下命令进行编译：

```bash
g++ -std=c++11 rpn_calc.cpp -o rpn_calc
